Quick Start Guide

Copy Master Pro is a powerful browser extension that enhances your copying and content management capabilities.

1. Install Extension

Load the extension in Chrome's developer mode or install from the Chrome Web Store.

2. Click Extension Icon

Click the extension icon in your toolbar to open the main panel.

3. Select & Copy

Select text on any webpage, then click any feature button to process and copy.

ğŸ’¡ Tip: Most features work with selected text. If no text is selected, some features will open a picker to let you choose content.

Snippet Variables

Use these variables in your snippets for dynamic content:

{date} - Current date (MM/DD/YYYY)
{time} - Current time (HH:MM:SS)
{host} - Current website domain
{title} - Current page title
{url} - Current page URL

File Naming Tokens

Available tokens for custom file naming:

{title} - Page title
{title:slug} - Slugified title
{host} - Domain name
{date:YYYY-MM-DD} - Custom date format
{time:HHmmss} - 24-hour time
{w}x{h} - Width x Height
{ext} - File extension
{seq} - Auto-increment number
